# Editor Support

Install the official **Kallo** extension for Visual Studio Code to get first-class support for `.kal` single-file components:

- **Syntax highlighting** for `<Server>`, `<Client>`, `<View>`, `<Style>`, `<Head>` blocks, with embedded TypeScript, CSS, and HTML.
- **IntelliSense** — completion for `$keyword` APIs, template tags, and directives.
- **Diagnostics** — structural validation and CSS checks as you type.
- **Snippets** for pages, layouts, components, and common blocks.

Search for **“Kallo”** in the VS Code Extensions panel, or install from the command line:

```bash
code --install-extension kallojs.kallo-vscode
```

The `.kal` file icon and language association are set up automatically.
